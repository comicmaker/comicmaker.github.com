Compatibility

Android 2.3 and above

Install to your android phone

Method 1: Android SDK must be installed on your machine inorder to install via adb command

1. Download the apk file here
2. Plug in your phone to your machine
3. adb install comicmaker.apk

Method 2: Install from Unknown source must be enabled in Settings of Android devices

1. Download the apk file here
2. Plug in your phone to your machine
3. Copy the apk over to phone's storage
4. Install the apk using a file manager
